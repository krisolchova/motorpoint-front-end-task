# Next.js + Jest

This example shows how to configure Jest to work with Next.js.

This includes Next.js' built-in support for Global CSS, CSS Modules and TypeScript.

## Node version

```bash
v16.15.1
```

## YARN version

```bash
v1.22.19
```

## NPM version

```bash
v8.11.0
```

## How to Use

In your terminal, run the following command:

```bash
npm run dev
```

```bash
yarn run dev
```

## Run Jest Tests

```bash
npm run test
```

```bash
yarn run test
```
